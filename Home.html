INTRODUCTION

What is C#?

C# is an elegant object oriented programming language design to work inside Microsoft .Net Framework. It provides simple programming by using Extensible Markup Language (XML) and Simple Object Access Protocol (SOAP) that allow access to an object or method of programming without requiring the programmer to write additional code for each stage.

C# programming allows developers to build cross-platform applications for Windows, Web, and mobile platforms. C# apps can be also deployed on Linux, Windows, iOS, and Android operating systems. It can be used to develop all kind of applications including Mobile applications, Desktop applications, Web applications, Web services, Web sites, Games, VR, Database applications and much more.

ENVIRONMENT

The .Net Framework

.Net framework is a software platform designed and developed by Microsoft. In plain terms, this is a virtual machine for compiling and running programs written in various languages such a C++, C#, Visual Basic, Jscript etc. Whereas all these languages can communicate with each other and access their framework. This helps you to write the following types of applications − Form-based applications, Web-based applications, and Web services

The two major components of .NET Framework are the Common Language Runtime and the .NET Framework Class Library.

  • Common Language Runtime(CLR) is the .NET Framework 's basic and virtual machine part. This is basically the execution engine that handles running the application. This provides support like thread management, garbage collection, type-safety, exception handling, and more.
  • Framework Class Library is a set of reusable, object oriented class libraries and methods and so on which can be implemented on CLR. The Class library provides set of APIs for reading and writing files, connecting to databases, drawing and more.

Integrated Development Environment (IDE) for C#

An IDE (Integrated Development Environment) is the software application that is used to type and compile your code. Visual Studio is the professional IDE developed by Microsoft that is used to design and develop the .NET applications. This is the best IDE for C# developers since the program, the framework and the language are all created by Microsoft. Visual Studio offers three different IDEs;

  • Visual Studio Enterprise
  • Visual Studio Professional
  • Visual Studio Community.

IDENTIFIERS

An identifier is a term used to identify a certain, class, variable, function or any user-defined items.

Rules for creating identifiers;

  • It must be unique.
  • It can contain only letters,digits and underscore “_” sign.
  • It should begin with a letter
  • It can’t contain reserved keywords.
  • It is case sensitive ( “myNum” and “mynum” are different variables)

Example:

   public class Program
   {
static void Main()
{
string Name;
}
}

In this example, the total number of identifiers is three (3);

  1. Program : Class Name
  2. Main : Method Name
  3. Name : Variable Name

C# VARIABLES

Variables are containers for storing your data values. Each variable in C# has a specific data type, which determines the size and layout of the memory of the variable, the range of values that can be stored in the memory, as well as the set of operations that can applied to it. In C# the basic values types are as follow;

  • Integral
  • Floating
  • Boolean
  • Decimal
  • Nullable
